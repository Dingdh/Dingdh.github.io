PG状态机

分布式存储系统的主要优势在于

  1. 在容量和性能上能够横向扩展
  2. 通过副本或者纠删码，提供高可用

如果是通过副本来提供高可用机制，那么就有一个基本问题，如何保证多个副本是一致的，否则如果读取不同副本得到的返回是不一样的，这样的存储系统是不可用的。
在RADOS里，副本的基本是PG(Placement group), 一个PG内包含一组Object，并且PG由一组OSD来提供服务(服务的OSD由CRUSH算法来确定)
